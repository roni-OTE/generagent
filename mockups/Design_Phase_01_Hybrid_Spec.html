The Hybrid: Anthropic Terminal × Linear Glow

בסיס B (Claude Code DNA) + עומק פרימיום מ-A. נשמע פשוט — אבל יש כאן 4 החלטות מרכזיות שכדאי לאשר לפני שעוברים לצבעים

codename: "forge"

למה היברידי, ולמה בדיוק היחס הזה

B לבד הוא מינימליסטי מדי — נראה כמו עוד CLI tool. A לבד מבדל אותך מ-Claude Code יותר מדי — סותר את הבקשה. ההיברידיות נכונה אם מקפידים על אחוזים: 80% B, 20% A. הבסיס (פלטת הצבעים, הטיפוגרפיה, הקומפוננטות, הריווח) הוא לחלוטין Anthropic Terminal. ה-20% של A נכנס רק ב-4 רגעים ויזואליים מוגדרים: ambient glows ב-Landing, glassmorphism במודאלים, indigo כצבע "AI thinking" (טעינה, ניתוח), ו-cinematic easing בטרנזישנים.

מה מ-B, מה מ-A

החלוקה המדויקת — שום דבר לא "התערבב במקרה"

★ 80% FROM DIRECTION B
  • פלטת בסיס: #0A0A0A · #141414 · #FAFAFA
  • Accent ראשי: Anthropic Orange #D97757
  • Typography: Heebo (HE) + Inter (EN) + JetBrains Mono
  • קומפוננטות: כפתורים, קלטים, כרטיסים — minimal flat
  • Border radius: 8-10px (לא 16)
  • בוט UI: monospace bubbles, terminal chrome
  • נגישות: WCAG AAA target
✦ 20% FROM DIRECTION A
  • Ambient glows: רק ב-Landing + Hero sections
  • Glassmorphism: רק במודאלים ובדרופ-דאונס
  • Indigo #5E6AD2: צבע משני ל-AI "thinking" moments
  • Cinematic easing: cubic-bezier(0.16, 1, 0.3, 1) בטרנזישנים
  • Subtle glow: מאחורי כפתור primary בלבד

Voice & Tone

איך GenerAgent מדבר עם המשתמש — בעברית

✓ DO — חבר מקצועי
  • ישיר וקצר "בוא נתחיל. ספר לי מה אתה עושה."
  • מסביר את עצמו "שאלה זו תעזור לי להבין X"
  • לא מתנצל יותר מדי "לא הבנתי, נסה לנסח אחרת" (לא: "מצטער מאוד...")
  • משתמש בלשון פעילה "בוא ננתח" (לא: "ייעשה ניתוח")
✗ DON'T — בוט מתחנף
  • רגשי או מתלהב לא: "מדהים! איזה תפקיד מעניין!"
  • מבטיח מה שאי אפשר לא: "אני אבנה לך סוכן מושלם"
  • ז'רגון טכני ללא הסבר לא: "MCP server עם stdio transport"
  • אמוג'ים בגוף השיחה רק אייקונים SVG/Unicode במידה
