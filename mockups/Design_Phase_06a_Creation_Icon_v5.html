למה זה עובד טוב יותר מהידיים
1. דפוס מוכר. משתמש שראה ChatGPT/Claude/Perplexity יודע מיד "המכונה חושבת". אין צורך ללמד metaphor חדש.

2. Pure CSS — אפס תלות. אין SVG path מורכב, אין GSAP, אין assets חיצוניים. כל האנימציה היא CSS conic-gradient + radial-gradient + transform.

3. Aurora native. ה-orb משתמש בדיוק בצבעי הפלטה: indigo במרכז, magenta בהילה, cyan בחלקיקים, warm cream ב-highlight. כמו אחות צעירה של ה-blobs ברקע של ה-Landing.

4. Scales perfectly. מ-32px (status indicator) ועד 480px (hero). אותו קוד, רק שינוי font-size/inset.

5. WCAG-friendly. respects prefers-reduced-motion, אין flicker, ה-orb נשאר נראה גם כשהאנימציה כבויה.

אם בעתיד תרצה לחזור למטאפורת הידיים — אפשר. אבל לדעתי הזה הדרך הנכונה ל-MVP.
