01 עקרונות

החוקים שמכתיבים כל אנימציה במערכת. אם הפרת את אחד מהם — בטל את האנימציה.

→ 01
תכליתית, לא דקורטיבית
כל תנועה אומרת משהו — אישור, ניווט, חזרה, רגש. אם היא רק "יפה" — מוחקים.
→ 02
מהירה, לא ממהרת
100-300ms לרוב האינטראקציות. 500ms+ רק לרגעי "מעבר סצנה". לעולם לא תחת 100ms — נראה buggy.
→ 03
expo-out כברירת מחדל
cubic-bezier(0.16, 1, 0.3, 1) — תחושה סינמטית של "התיישבות". משמש ב-80% מהמעברים.
→ 04
רק transform + opacity
אסור לאנמת width, height, top/left. GPU only — 60fps תמיד.
→ 05
prefers-reduced-motion קדוש
בכל אנימציה: או להאט מאוד (24s+) או להחליף ב-fade בלבד. לא לבטל — לשנות.
→ 06
stagger במידה
בקבוצה של 3+ — delay של 80ms ביניהם. פחות מ-3 — בלי stagger.

02 Easing curves

5 עקומות — לחץ על כל שורה כדי לראות את התנועה.

--ease-out-expo
ברירת מחדל — מעברים, hover, reveal. cubic-bezier(0.16, 1, 0.3, 1)
--ease-in-out
breathing, looping animations. cubic-bezier(0.4, 0, 0.2, 1)
--ease-out-back
toasts, success states, celebratory. cubic-bezier(0.34, 1.56, 0.64, 1)
--ease-linear
spinners, infinite rotation, conic gradients. linear
--ease-in
exit / dismiss אנימציות. cubic-bezier(0.4, 0, 1, 1)

05 Reduced motion

prefers-reduced-motion הוא קריטי. הכלל: מאטים או מחליפים — לא מבטלים.

!
אסור animation: none !important; על כל ה-DOM. המוצר ירגיש שבור (orb סטטי, סטייט לא משתנה).
!
נכון להאט אנימציות אמביינטיות: animation-duration: 24s במקום 4.5s. עדיין נראה חי.
!
נכון להחליף transforms ב-opacity בלבד: card hover ב-reduced — רק שינוי border-color, בלי lift.
!
בדוק עם: macOS Settings → Accessibility → Display → Reduce Motion. כל אנימציה צריכה עדיין להעביר את המסר.

06 ייצוא · CSS variables

העתק ל-globals.css. כל הקומפוננטות במערכת מושכות מכאן.

app/globals.css (motion tokens)
/* === MOTION === */ :root { /* easing */ --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1); --ease-in-out: cubic-bezier(0.4, 0, 0.2, 1); --ease-out-back: cubic-bezier(0.34, 1.56, 0.64, 1); --ease-in: cubic-bezier(0.4, 0, 1, 1); --ease-linear: linear; /* duration */ --dur-instant: 100ms; --dur-fast: 200ms; --dur-base: 300ms; --dur-slow: 500ms; --dur-cinema: 800ms; --dur-breath: 4500ms; } /* Global reduced motion handler */ @media (prefers-reduced-motion: reduce) { /* Ambient loops slow way down */ *:not(.essential-anim) { animation-duration: 24s !important; } /* Disable transform-based hovers, keep color changes */ *:hover { transform: none !important; } }
